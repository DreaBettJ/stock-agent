*生成时间: 2026-02-26*

## 案例：如何筛选行业龙头股

### 第一步：确定行业

以**新能源汽车**行业为例

### 第二步：筛选龙头

**龙头股特征：**
1. 市占率最高
2. 营收规模最大
3. 净利润最高
4. 行业定价权

### 第三步：量化筛选

```python
# 筛选条件示例
conditions = {
    "行业": "新能源汽车",
    "总市值": ">500亿",
    "ROE": ">15%",
    "营收增长率": ">20%",
}
```

### 第四步：验证基本面

| 指标 | 宁德时代 | 比亚迪 |
|------|----------|--------|
| 市值 | 1.2万亿 | 8000亿 |
| 营收增速 | +50% | +40% |
| 净利润 | 100亿 | 50亿 |
| ROE | 25% | 18% |

### 第五步：技术面确认

- 趋势向上
- 放量突破
- 均线多头排列
