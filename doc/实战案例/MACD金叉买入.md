# 技术分析实战：MACD金叉买入

*生成时间: 2026-02-26*

---

## 案例：MACD金叉买入法

### 什么是MACD金叉？

- **金叉**：DIF线从下向上穿过DEA线
- **买入信号**：金叉 + 零轴上方

### 实战筛选条件

```
✅ DIF和DEA在零轴上方
✅ 金叉时成交量放大
✅ 股价站上20日均线
✅ 前一波跌幅足够（>20%）
```

### 案例：某股票走势

```
日期       收盘价   MACD      信号
-----------------------------------------
1月1日    10.0     金叉      ⚠️ 观察
1月15日   11.5     零轴上方  ✅ 买入
2月1日    13.0     死叉      ⚠️ 卖出
```

### 实战步骤

#### 1. 筛选
- 跌幅 > 20% 的股票
- 底部横盘震荡

#### 2. 等待金叉
- DIF上穿DEA
- 成交量放大

#### 3. 买入
- 金叉确认后买入
- 止损设置在近期最低点

#### 4. 卖出
- 死叉卖出
- 或涨幅 > 15% 止盈
