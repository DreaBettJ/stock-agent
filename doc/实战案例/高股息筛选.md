# 高股息股票筛选实战

*生成时间: 2026-02-26*

---

## 案例：筛选稳定高股息股票

### 筛选条件

```
1. 股息率 > 3%
2. 连续分红 > 5年
3. 净利润 > 0（稳定盈利）
4. 市值 > 100亿
5. 行业龙头
```

### 实战筛选

| 股票 | 股息率 | 分红年限 | 行业 |
|------|--------|----------|------|
| 工商银行 | 4.5% | 20年 | 银行 |
| 建设银行 | 4.2% | 20年 | 银行 |
| 招商银行 | 3.8% | 15年 | 银行 |
| 中国神华 | 5.2% | 10年 | 煤炭 |
| 长江电力 | 3.5% | 15年 | 水电 |

### 买入时机

| 情况 | 建议 |
|------|------|
| 股息率 > 5% | 强烈建议买入 |
| 股息率 3-5% | 正常买入 |
| 股息率 < 3% | 观望 |

### 持有策略

```
持有期：至少1年以上
卖出条件：
  - 股息率 < 2%
  - 基本面恶化
  - 找到更好的替代标的
```
