# A股均线系统入门教程

## 一、什么是均线？

**均线，全称为"移动平均线"**，是把一段时间内的股票价格平均后连成的一条线。

### 打个比方

想象你每天记录体温，一周的体温平均值就是你的"5日均线"。如果记录一个月的平均值，就是"20日均线"。均线就是这样一根"平滑的趋势线"，能帮助我们看清价格波动的方向，而不是被每天的涨跌晃花眼。

---

## 二、各类均线的含义

### 📊 5日均线（周线）

- **计算方式**：最近5个交易日收盘价的平均值
- **特点**：**最灵敏**，能快速反映短期股价变化
- **适合**：短线交易者，观察短期趋势
- **比喻**：像体温的即时测量，波动很大

**举例**：某股票连续上涨，5日均线会很快向上翘头；一旦下跌，5日均线也会迅速反应。

---

### 📊 10日均线（半月线）

- **计算方式**：最近10个交易日收盘价的平均值
- **特点**：比5日均线**稍稳定**，仍属短期均线
- **适合**：观察短期支撑位和压力位
- **比喻**：像两周内的平均体温，比单周更稳定些

---

### 📊 20日均线（月线）

- **计算方式**：最近20个交易日收盘价的平均值
- **特点**：**非常重要！** 被很多机构作为操盘线
- **适合**：判断中期趋势的重要参考
- **比喻**：像月度健康报告，能反映一个月的身体状况

**⭐ 重点**：股价站上20日均线且均线向上，往往是**买入信号**；跌破20日均线且均线向下，可能是**卖出信号**。

---

### 📊 60日均线（季线）

- **计算方式**：最近60个交易日收盘价的平均值
- **特点**：**牛熊分界线**，非常重要！
- **适合**：判断中长期趋势
- **比喻**：像季度工作总结，能反映一个季度的整体状况

**⭐ 重点**：当股价站上60日均线，往往预示着**行情转暖**；当股价跌破60日均线，需要**警惕风险**。

---

### 📊 120日均线（半年线）

- **计算方式**：最近120个交易日收盘价的平均值
- **特点**：**长期趋势的认可线**
- **适合**：判断长期走势的支撑/压力位
- **比喻**：像半年成绩单，能反映半年的整体表现

---

### 📊 250日均线（年线）

- **计算方式**：最近250个交易日收盘价的平均值
- **特点**：**牛熊市的分水岭**，又称"牛熊线"
- **适合**：判断市场整体牛熊状态
- **比喻**：像年度总结，代表一年的整体水平

**⭐ 重点**：
- **股价在250日均线上方**：市场处于**牛市**状态
- **股价在250日均线下方**：市场可能处于**熊市**状态

## 三、均线的基本使用技巧

### 1️⃣ 均线交叉

**金叉（买入信号）**：
- **短期均线从下往上穿过长期均线**
- 例如：5日均线从下往上穿过20日均线
- **意味着**：短期价格开始走强，可能迎来上涨

**死叉（卖出信号）**：
- **短期均线从上往下穿过长期均线**
- 例如：5日均线从上往下穿过20日均线
- **意味着**：短期价格开始走弱，可能迎来下跌

**⭐ 重点**：周期越长，均线交叉的信号越可靠。**60日均线的金叉/死叉比5日均线更重要！**

**举例**：
```
案例：某股票走势
- 1月1日：5日均线从下穿过20日均线（金叉）→ 买入信号
- 1月15日：5日均线从上穿过20日均线（死叉）→ 卖出信号
```

---

### 2️⃣ 均线支撑和压力

**支撑位（股价的"地板"）**：
- 当股价下跌到某条均线附近时，**可能停止下跌，获得支撑**
- 均线就像一层"海绵"，能减缓下跌速度

**压力位（股价的"天花板"）**：
- 当股价上涨到某条均线附近时，**可能遇到阻力，难以突破**
- 均线就像一根"橡皮筋"，会把价格拉回来

**⭐ 重点**：**均线周期越长，支撑/压力越强！** 60日均线的支撑比5日均线更可靠。

**举例**：
```
案例：某股票从20元下跌
- 第一次跌到30日均线附近：获得支撑，反弹到22元
- 第二次跌到60日均线附近：获得更强支撑，反弹到21元
- 第三次跌到250日均线附近：获得最强支撑，反弹到20.5元
```

---

### 3️⃣ 均线多头排列 vs 空头排列

**多头排列（上涨趋势）**：
- **短期均线在 最上方，长期均线在 最下方，各条均线向上发散**
- 例如：5日 > 10日 > 20日 > 60日 > 120日 > 250日
- **意味着**：**强烈看涨！** 牛市特征

**空头排列（下跌趋势）**：
- **短期均线在 最下方，长期均线在 最上方，各条均线向下发散**
- 例如：5日 < 10日 < 20日 < 60日 < 120日 < 250日
- **意味着**：**强烈看跌！** 熊市特征

**⭐ 重点**：看到**多头排列**可以考虑**持股待涨**；看到**空头排列**要考虑**减仓或离场**。

## 四、实战应用案例

### 案例一：判断买入时机

**场景**：投资者小王想买入某只股票

**分析步骤**：
1. 先看250日均线：股价在年线上方 → 市场整体偏多
2. 看60日均线：股价站上60日均线，且60日均线向上 → 中期趋势向好
3. 等待回调：股价回调到20日均线附近，获得支撑
4. 确认信号：5日均线向上金叉10日均线

**结论**：这是一个**相对安全**的买入点

---

### 案例二：判断卖出时机

**场景**：投资者小李持有的股票开始下跌

**分析步骤**：
1. 5日均线死叉10日均线 → 短期走弱
2. 股价跌破20日均线 → 中期趋势可能改变
3. 60日均线开始走平并向下 → 警惕！
4. 均线开始出现空头排列 → 确认下跌趋势

**结论**：应该**考虑卖出或减仓**

---

### 案例三：震荡行情中的操作

**场景**：某股票在15-20元之间震荡

**操作策略**：
- **当股价跌到20日均线附近**：考虑**买入**（支撑位）
- **当股价涨到20日均线附近**：考虑**卖出**（压力位）
- **不恋战**：设置止盈止损，快进快出

## 五、注意事项 ⚠️

### ❌ 均线不是万能的

1. **均线有滞后性**：均线是"事后诸葛亮"，股价已经涨了，均线才发出信号
2. **震荡市失灵**：股价在均线附近反复波动，会产生很多假信号
3. **需要结合其他指标**：不要只看均线，要结合成交量、MACD等指标

### ❌ 新手常见错误

1. **频繁交易**：看到金叉就买，看到死叉就卖，结果亏手续费
2. **只看短期均线**：5日、10日均线波动太大，容易被"噪音"误导
3. **不设止损**：均线被跌破后心存侥幸，最后越套越深

### ✅ 正确做法

1. **用多条均线结合分析**：至少看20日、60日、250日三条线
2. **等待确认信号**：不要急于买入，等形态走稳
3. **设好止损位**：买入后以某条均线为止损线，比如跌破20日均线就卖出
