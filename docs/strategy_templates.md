# 股票投资策略模板库（长线/中线版）

## 0. 自由策略

### 策略描述
不预设固定规则，由代理根据当日市场数据、持仓状态和风险约束动态决策。

### 适用场景
- 你在探索期，希望先观察代理的自主分析与执行风格
- 市场风格频繁切换，固定模板约束过强

### 说明
- 允许代理自由组合趋势、动量、回调、分层等信号
- 仍需遵守账户资金与风控硬约束（仓位上限、止损等）

---

## 1. 均线趋势策略

### 策略描述
基于短期均线上穿/下穿长期均线的金叉/死叉信号进行交易。

### 买入条件
- 5日均线上穿20日均线（金叉）
- 成交量放大（今日成交量 > 昨日成交量 × 1.2）
- 股价位置：近20日涨幅 < 30%（不追高）

### 卖出条件
- 5日均线下穿20日均线（死叉）
- 或跌幅 > 7% 止损
- 或涨幅 > 15% 止盈

### 仓位规则
- 单只股票仓位 ≤ 20%
- 总持仓 ≤ 5只股票

---

## 2. 动量反转策略

### 策略描述
买入近期超跌的优质股票，反弹后卖出。

### 买入条件
- 近5日跌幅 > 10%（超跌）
- 市值 > 100亿（基本面稳健）
- PE < 30（估值合理）
- 行业龙头

### 卖出条件
- 涨幅 > 10% 止盈
- 跌幅 > 5% 止损

---

## 3. 行业动量策略

### 策略描述
跟随行业趋势，买入近期强势行业的龙头股。

### 买入条件
- 行业近20日涨幅排名前5
- 龙头股：行业市值最大
- 龙头股近5日涨幅 > 5%

### 卖出条件
- 行业跌破20日均线
- 行业近20日涨幅跌出前10

---

## 4. 突破新高策略

### 策略描述
买入突破历史新高的股票，强者恒强。

### 买入条件
- 股价创250日新高
- 成交量创250日新高（或者 > 平均2倍）
- 市值 > 50亿

### 卖出条件
- 跌破10日低点
- 跌幅 > 10% 止损

---

## 5. 龙头股趋势策略

### 策略描述
只做各行业的龙头股，趋势向上时持有。

### 龙头股定义
- 市值行业前3
- PE < 40
- ROE > 10%

### 买入条件
- 龙头股站上20日均线
- 20日均线向上

### 卖出条件
- 跌破20日均线
- 20日均线向下

---

## 6. 高股息策略

### 策略描述
买入高股息率的价值股，长期持有。

### 买入条件
- 股息率 > 3%
- PE < 20
- 连续3年分红

### 卖出条件
- 股息率 < 1.5%
- PE > 40

---

## 7. 龙头回调策略

### 策略描述
主线龙头股回调时买入，等二波。

### 买入条件
- 龙头股近期有涨停板（强势确认）
- 回调至10日均线附近
- 成交量萎缩（洗盘特征）

### 卖出条件
- 跌破20日均线
- 涨幅 > 20%

---

## 8. 题材趋势策略

### 策略描述
跟随市场热点题材，做趋势最强的股票。

### 买入条件
- 题材板块涨幅 > 5%
- 板块内涨停数 ≥ 3
- 选中龙头股（涨幅最大/最先涨停）

### 卖出条件
- 板块跌幅 > 5%
- 龙头股涨停打开
- 板块内跌停数增加

---

## 9. 激进策略

### 策略描述
在强势行情中优先追踪高动量标的，允许更快交易节奏和更高仓位利用率，强调“强者恒强 + 严格快止损”。

### 买入条件
- 当日或近3日处于主线板块（市场分层主线/龙头池）
- 技术面满足：`trend=up` 且 `macd=bullish`
- 突破/加速信号：近5日新高或放量上涨（成交额显著高于近5日均值）
- 非一字涨停，且有可成交流动性

### 卖出条件
- 跌破短期趋势（如5日均线/10日均线）
- 单笔浮亏达到止损阈值立即减仓或清仓
- 出现放量长上影、冲高回落等动能衰减信号
- 达到阶段止盈目标后分批兑现

### 仓位规则
- 单只股票仓位 ≤ 30%
- 总仓位上限可提高到 80%（仅在主线明确时）
- 采用分批进出（如 40% / 30% / 30%）

### 风险提示
- 激进策略仅适用于波段/题材主升阶段，回撤会显著高于保守策略
- 必须严格执行止损，不可逆势摊平

## 10. Agent提醒 + 手动执行策略（推荐默认）

### 策略描述
由 Agent 负责市场监控、信号筛选、风险提示与交易建议，人类交易者负责最终下单。

### 适用场景
- 希望保留主观判断，避免全自动执行风险
- 当前系统已具备分析能力，但不接券商交易闭环
- 重视可解释性与可控性

### 市场门控（先看环境再交易）
- `bull`：指数收盘 > MA20 且 MA20 > MA60，允许正常做多
- `sideways`：指数偏离 MA20 绝对值 < 2%，轻仓试错
- `bear`：指数收盘 < MA60，以观察为主，不新开仓

### 选股约束
- 剔除：`ST/*ST/退市风险/长期停牌`
- 流动性：近20日平均成交额 >= 5亿
- 波动约束：近20日振幅中位数 <= 8%
- 候选池：每次 5-10 只

### 入场触发（满足其一）
- 回踩确认：价格回踩 MA20 附近（±1.5%）后收阳，且成交量 > 5日均量
- 突破确认：突破近20日高点，且量比 >= 1.5
- 禁止追高：单日涨幅 > 7% 不追买

### 风险控制
- 单票仓位：10%-15%
- 组合总仓：`bull <= 70%`，`sideways <= 40%`，`bear <= 20%`
- 硬止损：-6%（收盘触发）
- 分批止盈：+12% 减1/3，+18% 再减1/3，剩余仓位用移动止损
- 组合熔断：当日回撤 >= 2.5% 或 周回撤 >= 5% 时，停止新开仓
---

## 策略选择建议

| 行情 | 推荐策略 |
|------|----------|
| 大涨趋势 | 均线趋势、突破新高 |
| 大跌反弹 | 动量反转、龙头回调 |
| 震荡整理 | 高股息、龙头股趋势 |
| 热点题材 | 题材趋势，行业动量，激进策略 |
| 人机协同 | Agent提醒 + 手动执行 |
| 长线价值 | 高股息、龙头股趋势 |

## 使用方式

创建 Session 时指定策略：

```bash
# 均线趋势策略
big-a-helper session create --name "均线策略" --prompt "你是均线趋势策略..." --mode simulation

# 高股息策略  
big-a-helper session create --name "高股息" --prompt "你是高股息策略..." --mode simulation

# 龙头股策略
big-a-helper session create --name "龙头股" --prompt "你是龙头股趋势策略..." --mode simulation
```
